About

Game programming graduate from NAIT College, he's currently seeking out new adventures in game design and programming. 

Intrigued by all things atmospheric, from foley audio and environment architecture to the most  basic of UI navigation.

With a keen feeling for when things are going right or wrong, he'll be the first to call out when things are deviating from their intended path.

See my previous experience on LinkedIn

WHAT I CAN DO FOR YOU

My current goal is to work within any  creative space. Preferably using what I studied about C# programming, game design, and VR technology.

- If you want playtesting, I can play that.
- If you want assets, I can code that.
- If you want SFX, I can make that.
- If you want people and schedules managed, I can manage that.
- If you want Oculus headset integration,  I can do all of that.

Although I am educated in C# programming within the Unity engine, I am willing to put in all of the effort needed to learn any skill required in order to work within your company.
